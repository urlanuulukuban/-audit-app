# 📋 Аудит Анкета - Система опросов

**Веб-приложение для проведения аудита с помощью интерактивной анкеты.**

---

## 🚀 Возможности

✅ **Защита паролем** - доступ только авторизованным пользователям  
✅ **Интерактивная викторина** - удобный интерфейс с вопросами  
✅ **Отслеживание прогресса** - визуальный прогресс-бар  
✅ **Подсчёт баллов** - автоматический расчёт результатов  
✅ **Скачивание результатов** - экспорт в текстовый файл  
✅ **PWA поддержка** - установка как приложение на телефон  
✅ **Оффлайн режим** - работает без интернета  
✅ **Адаптивный дизайн** - работает на всех устройствах  
✅ **Тёмная тема** - удобно для глаз

## 📱 Как использовать

### На компьютере:
1. Откройте файл `index.html` в браузере
2. Введите пароль: **12345**
3. Ответьте на вопросы
4. Посмотрите результаты

### На телефоне (iPhone):
1. Откройте ссылку в Safari
2. Нажмите **"Поделиться"** → **"На экран Домой"**
3. Приложение установлено! 🎉

### На телефоне (Android):
1. Откройте ссылку в Chrome
2. Нажмите **меню (≡)** → **"Установить приложение"**
3. Приложение установлено! 🎉

## 📊 Структура приложения

### Файлы:

| Файл | Описание |
|------|----------|
| **index.html** | Структура приложения (3 экрана) |
| **style.css** | Оформление и стили (тёмная тема) |
| **script.js** | Логика и функциональность |
| **manifest.json** | Конфигурация PWA |
| **service-worker.js** | Кэширование и оффлайн режим |

### Экраны:

1. **🔐 Экран входа** - ввод пароля
2. **📝 Экран викторины** - вопросы и ответы
3. **📊 Экран результатов** - итоги и статистика
